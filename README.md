#### **Sales & Interns Dashboard (Power BI)**
A business analytics dashboard built using Power BI that provides insights into sales performance and intern activity.

#### **Features**
- Data cleaning & transformation using Power Query  
- KPI indicators, bar charts, line charts, slicers, filters  
- Sales trend analysis (monthly/yearly)  
- Product-wise, category-wise breakdown  
- Intern performance & attendance tracking  
- Interactive visuals for better decision-making  

#### **Project Outcomes**
- Delivered a fully interactive dashboard enabling real-time analysis of sales performance.  
- Identified business insights such as revenue trends and top-performing products.  
- Improved decision-making through advanced filtering, drill-downs, and KPIs.  
- Saved time by automating data cleaning and transformation through Power Query.  
- Enhanced intern performance monitoring with clear metrics like attendance and contribution score.  
- Improved operational efficiency by providing a single consolidated view of performance.
